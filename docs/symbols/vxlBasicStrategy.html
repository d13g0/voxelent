Class vxlBasicStrategy

Implements a basic rendering strategy A vxlBasicStrategy object is selected by default as the strategy that an actor will use for rendering. A rendering strategy allows decoupling rendering specific code (i.e. code that access and communicates with the program) from the actor.
Defined in: BasicStrategy.js.
