<static> {quat4} quat4. fromAxes(view, right, up, dest)
Creates a quaternion from the 3 given vectors. They must be perpendicular to one another and represent the X, Y and Z axes. If dest is omitted, a new quat4 will be created. Example: The default OpenGL orientation has a view vector [0, 0, -1], right vector [1, 0, 0], and up vector [0, 1, 0]. A quaternion representing this orientation could be constructed with: quat = quat4.fromAxes([0, 0, -1], [1, 0, 0], [0, 1, 0], quat4.create());

Parameters:

{vec3} view

the view vector, or direction the object is pointing in

{vec3} right

the right vector, or direction to the "right" of the object

{vec3} up

the up vector, or direction towards the object's "up"

{quat4} dest

an optional receiving quat4
Optional

Returns:

{quat4} dest
